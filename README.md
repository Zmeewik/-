### 🥉 Хакатон «Леста Игры» — 3 место

**Проект:** 3D космический квест  
**Жанр:** Adventure / Puzzle  
**Движок:** Unity, C#

Игра про устранение неисправностей на космическом корабле через выполнение квестов и взаимодействие с объектами окружения.

#### Мои задачи:
- Управление командой разработки
- Проектирование игрового процесса и игровых механик
- Разработка структуры квестов и описание логики поведения объектов
- Программирование игровых сценариев на C#
- Реализация:
  - системы квестов
  - системы взаимодействия с объектами
  - звуковой системы
  - игровых событий
- Тестирование игровых механик
- Финальная полировка проекта перед презентацией

### Хакатон «Продуктовая И.Д.Е.Я.»  
**Организаторы:** Газпром нефть × ГУАП

**Проект:** 3D гиперказуальная релакс-игра  
**Жанр:** Hyper Casual / Simulation  
**Движок:** Unity, C#

Игра о повышении продуктивности сотрудников через необычную игровую механику взаимодействия с персоналом.

#### Мои задачи:
- Управление командой разработки
- Проектирование игрового процесса
- Описание логики поведения игровых объектов
- Разработка игровых сценариев на C#
- Тестирование и настройка игровых механик
